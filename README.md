### Run from terminal using
```bash
mvn clean test
```

```bash
mvn clean test -Dcucumber.filter.tags="@cleaning-sessions"
```

OR
```bash
mvn clean test -Dcucumber.options="src/test/resources/features/cleaning.feature"
```
